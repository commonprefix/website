Bifrost

Bifrost is a DeFi protocol on Polkadot providing cross-chain liquid staking services for multiple (para)chains. Common Prefix has performed a security audit of Bifrost's XCM Actions smart contracts. These contracts allow users of Polkadot's Astar and Moonbeam parachains to interact with Bifrost. Cross-chain communication is possible only if the Substrate address that corresponds to a given EVM address is known. The procedure of transforming one address format to the other requires the use of the Blake2b hash function. Common Prefix has also audited Bifrost's implementation of the Blake2b function.
